# 🏠 Home — 학습 데이터 저장소

> [!abstract] 이 보관함은
> **MSA 스타터 템플릿 개발**과 그에 필요한 **인증·데이터베이스 학습**을 정리하는 개인 지식 저장소(second brain)다. 모든 정리본은 아래 세 트랙으로 연결된다.

---

## 🧭 트랙 지도

### 🎯 MSA 템플릿 개발 — `msa/`
새 서비스를 추가하면 인증·게이트웨이·관측·배포가 자동으로 붙는 확장 가능한 MSA 스타터 템플릿. 그 위에 첫 서비스(PMS)를 올린다.
- **[[내 목표]]** — 목표·아키텍처·구성요소 허브
- **[[msa-roadmap]]** — Phase 0~6 마스터 로드맵
- [[arms-engine-분석]] — 실제 운영 코드(ARMS) 레퍼런스

### 🔐 인증/인가 학습 — `인증 학습/`
PMS를 관통 프로젝트로, 같은 앱을 v1 세션 → v2 JWT → v3 Keycloak으로 마이그레이션하며 익힌다.
- **[[인증 로드맵]]** — 0~14단계 마스터 로드맵 + 진행 상황
- 단계별: [[0단계. Spring Security 기초]] · [[1단계. Session Login]] · [[2단계. 세션의 한계와 JWT의 등장]] · [[3단계. JWT 직접 구현]] · [[4단계. 토큰 보안 (Access·Refresh)]] · [[5단계. Spring Security 심화]] · [[인증 6-7단계 OAuth2-OIDC 로그인]]

### 🗄️ 데이터베이스 학습 — `데이터베이스 학습/`
"왜 느려지고 어떻게 확장하는가"를 판단할 수 있게.
- **[[DB 학습 로드맵]]** — SQL → 정합성 → 성능 → 저장 구조 → 수평 확장 (13단계)

---

## 📍 현재 위치
- 인증 **0~7단계 완료** → 8단계(Keycloak) 진입 직전
- MSA **[[phase0-week-breakdown|Phase 0]]** (Keycloak + Gateway JWT 검증) — 인증 트랙과 합류 지점
- DB 트랙은 [[phase2-week-breakdown|Phase 2]]에서 템플릿에 붙여 실습 예정
